# Customer_behaviour_analysis
Data analytics project showcasing customer behaviour analysis using python,, sql and power bi

**1. Project Overview**

This project demonstrates an end-to-end data analytics workflow, starting from raw data processing to generating business insights. The objective was to analyze the dataset, clean and prepare it for analysis, extract insights using SQL queries, and present the findings through a Power BI dashboard, analytical report, and presentation.

The project highlights practical skills commonly used by data analysts such as data cleaning, exploratory data analysis (EDA), SQL querying, and data visualization.

**2. Dataset**

The dataset used in this project contains business-related transactional data used to analyze performance, trends, and operational patterns.

Key characteristics of the dataset:

Contains multiple attributes related to sales and operations

Includes categorical and numerical variables

Used for trend analysis, performance evaluation, and business insights

**4. Project Workflow / Steps**
   
**1. Data Loading**

Imported the dataset into Python using Pandas

Performed initial data inspection and structure analysis

**2. Exploratory Data Analysis (EDA)**

Examined distributions and patterns in the data

Identified trends, relationships, and anomalies

Generated visualizations to better understand the dataset

**3. Data Cleaning**

Handled missing values

Corrected inconsistent data entries

Removed duplicates and irrelevant records

Standardized column formats

**4. SQL Analysis**

Imported the cleaned dataset into MySQL Server

Performed SQL queries to answer analytical questions

Used aggregation, filtering, and grouping operations to derive insights

**5. Dashboard Development**

Built an interactive Power BI dashboard to visualize key metrics

Designed charts and KPIs to highlight important insights and business trends

**6. Reporting & Presentation**

Created a structured analytical report summarizing findings

Designed a presentation using Gamma to communicate insights effectively



**5. Dashboard**

The Power BI dashboard includes visualizations such as:

Key performance indicators (KPIs)

Trend analysis over time

Category-wise and regional performance breakdown

Comparative analysis to highlight problem areas

The dashboard helps stakeholders quickly understand the overall performance and key insights from the dataset.
